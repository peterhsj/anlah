登記類

申請應備文件：

申請人身分證明文件如本所得以電腦處理達成查詢者，得免提出

登記類應附文件
申請項目 應必備資料文件 備註
所有權移轉登記
  1. 土地登記申請書。
  2. 登記原因證明書：
    • 契約書（申請登記時檢附正副本各一，正本依權利價值千分之一貼印花稅票）。
    • 法院拍賣之權利移轉證明書或和解調解筆錄（申請部分與權利移轉證明書一致者免附）。
    • 公產出售移轉證明書（應另附登記清冊）。
  3. 義務人印鑑證明。
  4. 申請人身分證明文件。
  5. 各項繳稅證明。
  6. 土地或建物所有權狀。
  7. 其他 法令規定應附文件。
 
抵押權設定登記
  1. 土地登記申請書。
  2. 他項權利設定，移轉或內容變更契約書。
  3. 身分證明文件。
  4. 印鑑證明。
  5. 權利書狀。
  6. 其他法令規定應附文件。
  
抵押權塗銷登記
  1. 土地登記申請書及登記清冊(權利拋棄或債務清償證明書或塗銷同意證明書載明申請標的者得免附登記清冊)。
  2. 義務人印鑑證明書。
  3. 身分證明文件。
  4. 權利拋棄或債務清償證明書或抵押權塗銷同意證明書。
  5. 他項權利證明書。
  
建物所有權
第一次登記
  1. 土地登記申請書
  2. 建物測量成果圖（附繪建物平面圖及位置圖）。
  3. 權利人身分證影本或戶口名簿影本（法人應附證明文件）或戶籍謄本。
  4. 建物使用執照或其他合法房屋證明文件。
  5. 全體起造人分配協議書。（依使用執照無法認定申請人之權利範圍及位置時，需檢附）。
  6. 使用基地證明文件（實施建築管理前建造之建物，無使用執照，而建物與基地非屬同一人所有者，應檢附）。
  7. 權利證明文件（申請人非起造人時應檢附）。
 經符合簽證資格之土地登記專業代理人簽證者免附義務人印鑑證明。
權利書狀補發登記
  1. 土地登記申請書及登記清冊(保證書或切結書載明申請標的者得免附登記清冊)。
  2. 印鑑證明。
  3. 身分證明文件。
  4. 滅失事實之保證書或切結書。
  
權利書狀換發登記
  1. 土地登記申請書。
  2. 身分證明文件。
  
更名或住所變更登記
  1. 土地登記申請書。
  2. 戶籍謄本（應有記載變更前後記事）。
  3. 土地、建物所有權狀或他項權利證明書。
  
繼承登記
  1. 土地登記申請書及登記清冊(遺產分割協議書載明申請標的者得免附登記清冊)。
  2. 繼承系統表（依民法一一三八、一一四四條）。
  3. 戶籍謄本（被繼承人死亡時之戶籍謄本，繼承人在之戶籍謄本）。
  4. 繼承權拋棄書。
  5. 遺產稅繳納或免稅證明書。
  6. 遺產分割協議書正、副本（辦理分割繼承時檢附正本並應依不動產價值總額千分之一貼足印花票）。
  7. 印鑑證明書（辦理分割繼承時檢附之）。
  8. 土地或建物權狀或他項權利證明書。
  9. 其他法令規定應檢附文件。
  
更正登記
  1. 土地登記申請書及登記清冊。
  2. 身分證明文件。
  3. 更正登記原因證明文件。
  4. 土地、建物所有權狀或他項權利證明書。
